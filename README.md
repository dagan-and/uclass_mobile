# 개발 지침서 (Development Guidelines)

본 문서는 모바일(Android/iOS) 개발 시 준수해야 할 기준과 가이드라인을 제시합니다.

## 🤖 Android 개발
### 개발 환경
- **Android API**: 23 이상 (Android 6.0+)
- **언어**: Kotlin 우선, Java 보조
- **빌드 도구**: Gradle 8.9
- **Java 버전**: Java 11


## 🍎 iOS 개발
### 개발 환경
- **iOS**: 15 이상
- **언어**: Swift 우선 (5.9+), Objective-C 보조
- **도구**: Xcode 최신 버전
- **패키지 관리**: POD
